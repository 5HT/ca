SYNOPSIS

Open Source Certificate Authority (CA) by SYNRC. It provides Public Key Infrastructure (PKI) with both: Elliptic Curve Cryptography (ECC), best suited for IoT security; and legacy Rivest–Shamir–Adleman (RSA) cryptography. It also supports CSR certificate enrollment and provides OCSP responder for fast revokation checking. As a dependency it contains LDAP directory server.

SPEC

  • OCSP
  • LDAP
  • RSA
  • ECC
  • CSR
